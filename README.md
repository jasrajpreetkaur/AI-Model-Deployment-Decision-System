# AI-Model-Deployment-Decision-System
##A decision making system using Python that evaluates AI models and make decisions based on some parameters like accuracy, precision, recall, F1 score, Inference time and Memory usage and tell whether the model is ready for production or not.
##Parameters used:
-Accuracy
-Precision
-Recall
-F1 Score
-Inference time
-Memory usage
##Decision Criteria
The Model is considered Production Ready if
-Accuracy>=90%
-Precision>=90%
-Recall>=90%
-F1-Score>=90%
-Inference Time<=100ms
-Memory usage<=512MB
